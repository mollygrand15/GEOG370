Methods

To create this map, I merged the spatial zip code boundary layer to the table demographic statistics layer in R, and then wrote this table out to spatial layer. I then imported it into ArcGIS Pro, and created a chloropleth map to show the black population by zip code in NYC. I then mapped the public parks polygon layer.
